Listas

As listas podem ser ordenadas ou não ordenadas. A lista ordenada é definida pela tag <ol>. A lista não ordenada é definida pela tag <ul>. Os items são definidos pela tag <li>.

Exemplo

Lista Ordenada

<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
<li>Item 4</li>
</ol>

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Lista Não Ordenada

<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
<li>Item 4</li>
</ul>

  • Item 1
  • Item 2
  • Item 3
  • Item 4
